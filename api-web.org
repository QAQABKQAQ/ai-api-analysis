提示价格：$5.000000 / 1M tokens

提示 3 tokens / 1M tokens * $5.000000 + 缓存 10381 tokens / 1M tokens * $0.500000 (倍率: 0.1) + 1h缓存创建 24709 tokens / 1M tokens * $10.000000 (倍率: 2) + 补全 3214 tokens / 1M tokens * $25.000000 * 分组倍率 0.75 = $0.249484
