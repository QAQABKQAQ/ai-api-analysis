* 📋 概述

本报告基于 api-web.org 中收集的数据，对比分析各第三方 API 服务商提供的 Claude Code (Opus 4) 使用价格。
核心指标为 *1 CNY 可兑换多少 USD 的 Claude 官方等价额度* ，该值越高表示性价比越好。

** 参考基准
- *Claude Opus 4 官方 API 价格*:
  - 输入: $15 / 1M tokens
  - 输出: $75 / 1M tokens
  - 缓存写入: $18.75 / 1M tokens
  - 缓存读取: $1.875 / 1M tokens
- *当前汇率*: 1 USD ≈ 6.858 CNY (基于 aicoding 充值数据推算)

- 整体倍率约 *5.26 ~ 6.05 倍* 官方价格
